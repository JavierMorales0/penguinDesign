Fonts

Fuentes nativas

penguinDesign maneja fuentes que se adaptan perfectamente con la temática elegante, moderna y simple que se ha implementado en la misma de manera precisa

Entre las variaciones de fuentes que logramos encontrar con este framework se encuentran:

  • Normal
@font-face { font-family: "normal"; }
  • Light
@font-face { font-family: "light"; }
  • Bold
@font-face { font-family: "bold"; }
  • ExtraBold
@font-face { font-family: "extraBold"; }

El font-family utilizado en el framework se aplica de manera nativa en todo el documento como un global font, puede cambiarse esto desde su propio font-face asociado como observamos anteriormente.



Bold y ExtraBold

Para remarcar cierto contenido, agregando más relleno que el texto normal. Tenemos dos variaciones en cuando al texto bold, por un lado tenemos .font-bold para un texto con resaltado normal, y por otro el .font-extraBold para un texto con más grosor:

Párrafo utilizando bold en una palabra.


<div class="container bg-light-secondary">
  <div class="row">
    <div class="float-center">
      <p>Párrafo utilizando <span class="font-bold">bold</span> en una palabra.</p>
    </div>
  </div>
</div>

Párrafo utilizando extrabold en una palabra.


<div class="container bg-light-secondary">
  <div class="row">
    <div class="float-center">
      <p>Párrafo utilizando <span class="font-extraBold">extrabold</span> en una palabra.</p>
    </div>
  </div>
</div>
