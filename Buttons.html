Buttons

Es el elemento crea botones marcadores.

La etiqueta de HTML <button> representa un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar y simple de aplicar

Ejemplos Simple

Sintaxis

        <button name="button">click aca</button>

Etiquetas de buttons

Ejemplo

Sintaxis

        <button class="btn btnColor-white">White</button>
        <button class="btn btnColor-redt">Red</button>
        <button class="btn btnIColor-blue">Blue</button>

Botones Mas Grandes

Ejemplo

Sintaxis

          <button class="btn widht-25 btnColor-red">Grande Red</button>
          <button class="btn widht-25 btnIColor-blue">Grande Blue</button>

Botones con enlace

Ejemplo

link link

Sintaxis

          <a class= "btn btnColor-red" href="#" role="button">link</a>
          <a class= "btn btnIColor-blue" href="#" role="button">link</a>

Mas estilos de botones

Ejemplo

Sintaxis

        <button class= "btn-circle-add btnIColor-white"></button>
        <button class= "btn-circle-delete btnColor-green"></button>
        <button class= "btn-circle-check btnIColor-red"></button>
        <button class= "btn-circle-alert btnColor-yellow"></button>
